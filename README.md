## My Personal Website
Welcome everyone, this is the beautiful code for my [website](https://cedomir.tech)

## How to run
It's a simple one :)
```
cd <your favorite workspace>
git clone https://github.com/cedomir-spalevic/cedomir
cd cedomir
npm run setup
npm run dev
```
